# Python calculator

## Description
A simple CLI calculator in Python. The script ask two numbers and an operator 
among [+, -, *, /] to make some calculations.

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Input validation with error handling
- Calculation history (viewable on demand)
- Exit anytime with 'q' command
- Formatted output with 2 decimal places

## Installation
Clone the repository:
```bash
git clone https://github.com/AlexandreCst/python-calculator.git
cd python-calculator
```

No additional dependencies required (uses Python standard library only).

## How to Use
Run the program:
```bash
python calculator.py
```

Follow the prompts:
- Enter your first number
- Enter your second number
- Choose an operation (+, -, *, /)
- View result
- Type 'q' anytime to quit

## Requirements
- Python 3.9 or above
